CATCH BLOCK

Introduction

A catch block is a group of statements, enclosed in braces { } which are used to handle
a specific exception that has been thrown. A catch block should be placed immediately
after the try block. When an exception occurs in a try block, program control is
transferred to the suitable catch block. A catch block is specified by the keyword
catch followed by a single argument within parenthesis ( ).


Syntax

The general syntax for a catch block is
try{
//code that may throw exception
}
catch(ExceptionType1 e) {
//exception handler for ExceptionType1
}
catch(ExceptionType2 e) {
//exception handler for ExceptionType2
}

An Example :

classtrycatch{
public static void main(String args[]) {
int x=50;
int y=0;
int z;
try{
z = x/y; System.out.println("z = "+z);
}
catch(ArithmeticException e) {
System.out.println(e);
System.out.println("Divide by Zero");
}
System.out.println("End of the Program");
}
}

Explaination to the above Program:

In the above program the class name is trycatch and it has three integer variables x, y, z. Inside the try statement, an attemptof divide by zero is made. It creates an exception object of type ArithmeticExceptionand throws it out. The statement after the exception will not be executed and the value of z will not be displayed.The control transfers to the catch statement directly.Catch block catches the exception and puts in the object e. While printing the value of e it prints the description of the exception. The the subsequent lines are executed, as the output "Divide by Zero" and "End of the Program" is displayed.

Multiple Catch Blocks

In some cases, more than one exception may occur in a single program. To handle this type of situation,
two or more catch statements can be used in a program. Each catch will catch a different type of
exception. When an exception is thrown, each statement is inspected in order and the first one
whose type matches that of the exception is executed. After one catch statement
executes, the others are bypassed and the execution continues after the try-catch
block.
